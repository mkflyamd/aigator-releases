Skills and Plugins
Browse, install, and create in Gator. What's in a skill?

A skill extends what AI Gator can do. Skills range from simple instructions to fully featured packages:

  • 📝 instructions — a SKILL.md file telling Gator how to handle a task.
  • 🔧 tools — Python functions (tools.py) Gator can call directly.
  • 🤖 agents — specialist sub-agents the skill can dispatch.
  • 🪝 hooks — event triggers fired at lifecycle points.
  • 🔌 MCP servers — Model Context Protocol servers bundled with the skill.

A skill with code, agents, hooks, or MCP servers is called a plugin in our docs. Gator treats both the same.
